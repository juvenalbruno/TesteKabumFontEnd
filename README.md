<h3>ℹ️ Como usar</h3>

    # Clone esse repositório
    $ git clone https://github.com/juvenalbruno/TesteKabumFontEnd.git
    
    # Instalar dependências do projeto
    $ yarn install
    
    # Executar o comando para rodar o servidor
    $ yarn server

    
    # Executar o comando para rodar o projeto
    $ yarn start
